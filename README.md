## Requirements

- PHP 8.1 or newer
- Composer 2.5.5 or newer
- MySQL 15.1 

## Installation

#### Instructions
- clone or download this repository
- change `.env.example` to `.env`
- create database and configure database name at `.env`

#### Install commands
- `composer install`
- `php artisan migrate`
- `php artisan key:generate`
